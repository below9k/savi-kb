Markdown Formatting

Markdown is a syntax for creating HTML structure (headings, lists, links, etc). Textbox.io automatically converts markdown into HTML as you type.

Creating Headings
Syntax # Heading
Trigger [enter]

Headings are created by starting a new line with 1-6 hash characters followed by a space and then the heading text, like: # Heading Text. The number of hash characters corresponds to heading levels 1-6. Pressing enter after the heading text will convert the markdown to HTML.

Creating Bulleted Lists
Syntax * text
Trigger [enter]

Bulleted lists are created by starting a new line with an asterisk and a space and then the first list item text, like: * list item. Pressing enter after the first list item text will convert the markdown to HTML and start the second list item.

Creating Numbered Lists
Syntax 1. text
Trigger [enter]

Numbered lists are created by starting a new line with a number, a period, a space, and then the first list item text, like: 1. list item. Pressing enter after the first list item text will convert the markdown to HTML and start the second list item.

Apply Emphasis to Text (Italics)
Syntax *emphasis*
Trigger [space]

Emphasis is be applied by wrapping text in asterisk and then pressing space, like: *text*.

Apply Bold to Text
Syntax **bold**
Trigger [space]

Bold is applied by wrapping text in double asterisk and then pressing space, like: **text**.

Create a Horizontal Rule
Syntax ---
Trigger [space] or [enter]

Create a horizontal rule by entering 3 or more dashes on a new line, then press the return key, like: ---.
