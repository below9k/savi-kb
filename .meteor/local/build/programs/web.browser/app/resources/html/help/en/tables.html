Tables
Inserting a Table

To add a table to a document, select the Insert menu and move to the Tables item. Move the mouse over the table grid to choose the desired numbers of rows and columns and then click to create the table. To add a header row or column use the Header Row toggle or Header Column toggle .

Modifying Table Structure

Modify the structure of a table in a document by right-clicking the table to show the Table Modification menu. From the Table Modification menu, columns/rows can be added or deleted, and rows/columns can be converted to headings.

Resizing Table Columns

To resize a table column, click a column border and drag the column to the left or right.
