Formatting Text
Paragraph Styles and Headings

The Paragraph Styles and Headings menu displays the style of your text selection and lets you change the desired paragraph style or heading. Click the menu to show a list of available options based on your selection. Choose a new option by selecting the menu item from the list.

Change Font

The Formatting menu includes a list of fonts you may apply to selected text and displays the current text font. Click the Formatting menu and move to the Font menu item to see the list of available fonts. Choose a new font for your selection by clicking an option.

Change Font Size

To change the size of the currently selected text, open the Formatting menu and move to the Size item. You may increase/decrease the font size with the and button, or enter it directly in the input provided. To change the font-size using the input field, add the desired font value and unit of measure, then press enter.

Change Text Color

Open the Formatting menu and move to the Color item to change the text color and text highlight (background) color of selected text.

Change Alignment

The Alignment menu displays the current alignment value for the selected text and lets you modify the alignment of the current selection.

Make a list

Any line may be converted to a list by selecting the Bulleted List item or the Numbered List item .

Indent Text

Indentation can be added to the selected text by clicking the Indent item . Indentation can be removed by clicking the Remove Indent item .

Make a quote

A quote (blockquote) can be applied to a selection by clicking the Blockquote item . To remove the quote, click the Blockquote item again.

Remove Formatting from Text

Remove formatting from text by clicking the Remove Format item .
