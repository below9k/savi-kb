Re-Using Snippets

Snippets are small bits of text you save for reuse later.

You can create a snippet with the Snippet dialog, and insert that snippet into any future document either from the Snippet dialog or using a Snippet keyboard macro.

Using the Snippet Dialog

The Snippet dialog lets you insert, create, edit, and delete your snippets. Launch the Snippet menu with the button or by pressing CTRL + J or âŒ˜ + J.

Search Search through your saved snippets with the search bar in the upper left of the Snippet dialog (upper left).
Preview Preview snippets by clicking on the arrow to the right of snippet's name in the list. To edit a snippet you are previewing, click the edit button in the upper right of the window.
Create Create a snippet by selecting the new snippet button in the lower left.
Insert Insert a snippet by clicking its name in the list, or by previewing it and clicking the insert button.
Using Snippet Keyboard Macros

You can insert a snippet into any Textbox.io editor by typing the snippet name inside double braces and then pressing space or enter, like: {{name}}.
