Inserting Images, Links, & Media
Insert an image

To insert an image into the document, select the Insert menu and click the Insert Image item . The resulting Insert Image dialog can be used to choose images from your local computer or add an image from a web URL.

Insert a link

Insert a link into the document at the current cursor position by selecting the Insert menu and clicking the Insert Link item .

Modify a Link

To modify a link in a document, click the link to show the Link dialog. The link dialog allows you to view the target of the link, edit the link, or remove the link from the document.
